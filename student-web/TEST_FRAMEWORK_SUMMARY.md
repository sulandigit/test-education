## 4. 测试命令和脚本

### 单元测试和集成测试
```bash
# 运行所有测试
npm run test

# 运行测试并查看UI界面
npm run test:ui

# 运行测试一次（CI模式）
npm run test:run

# 运行测试并生成覆盖率报告
npm run test:coverage
```

### 端到端测试
```bash
# 安装Playwright浏览器
npm run install:playwright

# 运行E2E测试
npm run test:e2e

# 运行E2E测试并查看UI界面
npm run test:e2e:ui

# 运行E2E测试（可视化模式）
npm run test:e2e:headed
```
