## 🛠 技术架构

### 前端技术栈
- **Vue.js 3**: 采用Composition API的现代化前端框架
- **Nuxt.js 3**: SSR/SSG框架，提供优秀的SEO和性能
- **TypeScript**: 类型安全的JavaScript超集
- **Pinia**: 现代化的Vue状态管理库
- **Element Plus**: 企业级UI组件库
- **Sass/SCSS**: CSS预处理器

### 开发工具
- **Vite**: 快速的构建工具
- **ESLint + Prettier**: 代码规范和格式化
- **Vitest**: 现代化的单元测试框架
- **Playwright**: 端到端测试框架

## 📁 项目结构

```
student-web/
├── components/          # 可复用组件
│   ├── Header.vue      # 页面头部
│   ├── Footer.vue      # 页面底部
│   ├── CourseCard.vue  # 课程卡片
│   └── VideoPlayer.vue # 视频播放器
├── pages/              # 页面组件
│   ├── index.vue       # 首页
│   ├── course/         # 课程相关页面
│   ├── learning/       # 学习页面
│   ├── auth/           # 认证页面
│   └── user/           # 用户中心
├── stores/             # 状态管理
│   ├── user.ts         # 用户状态
│   ├── course.ts       # 课程状态
│   └── learning.ts     # 学习状态
├── api/                # API服务层
│   ├── auth.ts         # 认证API
│   ├── course.ts       # 课程API
│   └── learning.ts     # 学习API
├── middleware/         # 路由中间件
├── utils/              # 工具函数
├── assets/             # 静态资源
├── tests/              # 测试文件
│   ├── components/     # 组件测试
│   ├── stores/         # 状态测试
│   ├── integration/    # 集成测试
│   └── e2e/            # 端到端测试
└── docs/               # 项目文档
```
